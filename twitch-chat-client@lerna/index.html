Twitch.js Chat Client

Welcome to the documentation of the Chat module of Twitch.js, a library for NodeJS and the browser that aims to provide an easy interface to all of Twitch's programmatically available features.

Please feel free to browse the menu on the left to get started with the library and to check out all the different available classes.
