Twitch.js PubSub Client

Welcome to the documentation of the PubSub module of Twitch.js, a library for NodeJS and the browser that aims to provide an easy interface to all of Twitch's programmatically available features.

Please feel free to browse the menu for the left and either get started with the library or check out all the different available classes.
