TwitchClient

The main entry point of this library. Manages API calls, chat and PubSub connections and the use of access token in all of these.

Constructor

new TwitchClient(config)

Creates a new Twitch client instance.

ParameterTypeRequiredDefaultDescription
configPartial<TwitchConfig>none

Configuration for the client instance.

Methods

apiCall(options)

Makes a call to the Twitch API using your access token.

ParameterTypeRequiredDefaultDescription
optionsTwitchApiCallOptionsnone

The configuration of the call.

Return type: Promise<T>

getChatClient(identifier)

Creates a chat client with your credentials.

ParameterTypeRequiredDefaultDescription
identifierstring"default"

The identifier of the chat client.

Passing different strings to this will create separate clients. Passing the same string twice will return the same client.

Return type: Promise<ChatClient>

getPubSubClient(identifier)

Creates a PubSub client with your credentials.

ParameterTypeRequiredDefaultDescription
identifierstring"default"

The identifier of the PubSub client.

Passing different strings to this will create separate clients. Passing the same string twice will return the same client.

Return type: SingleUserPubSubClient

getTokenInfo()

Return type: Promise<TokenInfo>

ParameterTypeRequiredDefaultDescription
optionsTwitchApiCallOptionsnone

The configuration of the call.

clientIdstringnone

The client ID of your application.

accessTokenstringnone

The access token.

Return type: Promise<T>

getAccessToken(clientId, clientSecret, code, redirectUri)

Retrieves an access token with your client credentials and an authorization code.

ParameterTypeRequiredDefaultDescription
clientIdstringnone

The client ID of your application.

clientSecretstringnone

The client secret of your application.

codestringnone

The authorization code.

redirectUristringnone

The redirect URI. This serves no real purpose here, but must still match with the redirect URI you configured in the Twitch Developer dashboard.

Return type: Promise<AccessToken>

Retrieves information about an access token.

ParameterTypeRequiredDefaultDescription
clientIdstringnone

The client ID of your application.

accessTokenstringnone

The access token.

Return type: Promise<TokenInfo>

refreshAccessToken(clientId, clientSecret, refreshToken)

Refreshes an expired access token with your client credentials and the refresh token that was given by the initial authentication.

ParameterTypeRequiredDefaultDescription
clientIdstringnone

The client ID of your application.

clientSecretstringnone

The client secret of your application.

refreshTokenstringnone

The refresh token.

Return type: Promise<AccessToken>

withCredentials(clientId, accessToken, refreshConfig, config)

Creates a new instance with fixed credentials.

ParameterTypeRequiredDefaultDescription
clientIdstringnone

The client ID of your application.

accessTokenstringnone

The access token to do API requests with.

refreshConfigRefreshConfignone

Configuration to automatically refresh expired tokens.

configPartial<TwitchConfig> {}

Additional configuration to pass to the constructor.

Note that if you provide a custom authentication provider, this method will overwrite it. In this case, you should use the constructor directly.

Return type: TwitchClient
